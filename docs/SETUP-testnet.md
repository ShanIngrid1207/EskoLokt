# EskoLokt Stellar Testnet Setup Guide

Welcome to the EskoLokt prototype! Follow this short, friendly guide to set up your wallet and start testing escrow payments on the Stellar Testnet.

## Step-by-Step Setup

1. **Install a Stellar Wallet**
   Install the compatible Stellar wallet on your phone.
   > TODO(owner): wallet name + test-USDC issuer address

2. **Switch to Testnet**
   Open your wallet's settings and switch the network from Mainnet to **Testnet**. This ensures you are using test tokens and not real money.

3. **Get Test Funds**
   Within the EskoLokt app interface, locate and tap the **Get test funds** button. This will automatically fund your address with:
   - Test XLM (used to cover Stellar network transaction fees)
   - Test USDC (used for the prototype's escrow deposits)

4. **Approve the USDC Trustline**
   When the app prompts you, approve the **USDC trustline** request (requires one tap). On Stellar, a trustline is required before a wallet address can receive or hold any asset other than XLM.

5. **Start Transacting**
   You're all set! You can now open a seller's order link to lock a deposit or create your own order from the dashboard.
